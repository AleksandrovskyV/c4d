
Нужно довести скрипт в cinema 4d. Вот список правок:

1. Правка по интерфейсу:

При запуске - проверяем есть ли "render setting" с название mp4_drafts
Если его нет -  всплывает диалоговое окно c такими значениями:

Dialog Text: Write on ".\mp4_previews\preview_#.mp4"?
Answer Buttons: yes и custom

Если "yes" - создается "render setting" mp4_drafts и происходит процесс записи mp4 файлов в эту папку, уже реализованный в коде скрипта


Если "custom" - открывает браузер, чтобы указать папку и название (обычный указать пути и названия). 
После под капотом создается "render setting" выставленным пользователем и происходит процесс записи mp4 файлов в эту папку, уже реализованный в коде скрипта

После рендера, переключаемся на тот  "render setting" который был включен у пользователя до активации скрипта. 


2. Правка по основному коду:
Мы всегда ищем максимальную цифру после "_", и ставим +1 от неё. 
Если допустим мы удалили _1 или _2 он все равно должен идти от "5" если такой имеется в папке. 
Я заметил, что если я удалил какой _1 и _2 - он снова начала писать от 1 
Возможно чтобы ускорить поиск - можно просто посмотреть цифру последнего файла (я не знаю, даёт ли система узнать последний файл записанный на диск). 

Вот исходный код:
import c4d, os, re

def main():
    # 1. Получаем активный документ и настройки рендера
    doc = c4d.documents.GetActiveDocument()
    rdata = doc.GetActiveRenderData()
    path = rdata[c4d.RDATA_PATH]
    
    if not path:
        c4d.gui.MessageDialog("setpath")
        return
        
    # 2. SET MP4 (H.264)
    rdata[c4d.RDATA_FORMAT] = c4d.FILTER_MOVIE
    
    # 3. Работа с путями и поиск свободного индекса (_1, _2, _3)
    base_path, ext = os.path.splitext(path)
    ext = ".mp4"
    
    # Очищаем базовый путь от старых хвостов _1, _2, если они уже были прописаны
    base_path = re.sub(r'_\d+$', '', base_path)
    
    # Ищем на диске первый свободный номер, чтобы не перезаписать старый тест
    counter = 1
    new_path = f"{base_path}_{counter}{ext}"
    while os.path.exists(new_path):
        counter += 1
        new_path = f"{base_path}_{counter}{ext}"
        
    # 4. Прописываем итоговый путь с нужным индексом (без расширения, C4D добавит его сама)
    rdata[c4d.RDATA_PATH] = f"{base_path}_{counter}"
    
    # Обновляем интерфейс программы, чтобы изменения применились
    # Запуск рендера в Picture Viewer
    c4d.EventAdd()
    c4d.CallCommand(12099) 

if __name__ == '__main__':
    main()
