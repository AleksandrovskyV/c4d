Place the "vsky.scripts" folder into your Cinema 4D scripts directory

You can open this directory by going to: 
Cinema 4D R20: Top Menu > Scripts > User Script > Script Folder...
Cinema 4D R23: Top Menu > Extensions > User Scripts > Script Folder...

The resulting path should look like this:
...\AppData\Roaming\Maxon\Cinema 4D R23_########\library\scripts\vsky.scripts\
