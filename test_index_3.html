- version token. Uses active "Render Settings"

- номером итерации. Использует текущие
"Render Settings"...

* Linked Camera

Сamera linked to the active one via CameraMorph tag. Includes optional Xpresso to expand your render area while strictly maintaining the original focal length

Камера привязанная к активной, через CameraMorph тег. Включает возможность расширить зону рендера без искажения фокусного расстояния

Tested: R20

download

Features:

- Camera is created on the top level, ready for baking the original PSR animation data
- Ready-to-use Render Settings tailored for the expanded region via Xpresso tag
- The entire сamera setup can be completely removed along with Render Settings in one click

- Камера создаётся на верхнем уровне, что удобно для запекания исходной PSR-анимации
- Для расширенной области рендера через Xpresso тег создаётся готовый профиль Render Settings
- Опция для удаления добавленной камеры, включая профиль "Render Settings", в один клик
